# SimplePlatformer

A simple platformer built using Godot, which also uses the MIT License. This project is meant to display Godot's tools that a user might need to use in order to build a similar project for learning purposes.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

>* [Godot's Latest Version](https://godotengine.org/)
>* Follow Their Installation Instructions


### Installing

>1. First, download the zip of the codebase from the project Homepage.
>2. Extract the zip in a project directory of your choosing.
>3. Open Godot and click "Import" in the top left corner, and select the extracted folder.
>4. Open the project and enjoy testing!
